# Semantic Cube Architecture

---
本論文は、私の発案と、Gemini、Copilot、AI達との協業によって作成しています。

本論文は高難易度な部分があるため、必要に応じてAI要約を活用すると理解しやすくなります。
---

## 何これ
LLM単体で全てを行うAIモデルではなく、記憶・検索・判断をそれぞれの役割で分離したアーキテクチャを作ってみた。

## 何を解決するの？
- 長期対話でスレッドが崩壊する問題。
- 意見の平均化によるハルシネーション
- スレッド寿命の長大化

## どうやってるの
- 意味キューブ
- SRAG
- 引力マップ（判断）
- 記憶のα代謝
- 対話の進行ベクトル

## 一言
AIをLLM単一モデルから、AIシステムに強化させる試み。
